# 🏗️ روزنگار

چت‌بات هوشمند تلگرام برای مرتب‌سازی و طبقه‌بندی **گزارش روزانه‌ی کارگاه**.

سرکارگر در طول روز پیام‌های متنی و صوتی می‌فرستد؛ هوش مصنوعی (Google Gemini) داده‌ها
را استخراج و در جدول‌های منظم ذخیره می‌کند و آخر روز یک **گزارش استاندارد اکسل** تولید می‌شود.

---

## ✨ امکانات نسخه‌ی فعلی

- 🟢 دکمه‌ی **شروع روز / پایان روز** (ماشین حالت روزکاری)
- 💬 دریافت گزارش به‌صورت **متن یا ویس** (پیاده‌سازی صوت فارسی با Gemini)
- 🧠 استخراج ساختاریافته‌ی: **کارکرد نیرو، فعالیت‌ها، موانع، دوباره‌کاری**
- ⏱️ محاسبه‌ی خودکار **نفر-ساعت و اضافه‌کاری** (با تشخیص صبح/عصر از متن)
- 👷 دفترچه‌ی نیروها با ثبت خودکار
- 📊 گزارش اکسل استاندارد + ارسال به **کانال بک‌آپ**
- 🗂️ ذخیره‌ی **پیام خام** برای ردِ ممیزی

## 🧭 معماری

```
تلگرام ──webhook──▶ Next.js روی Vercel ──▶ Postgres (Neon)
                          │
    هر پیام → ذخیره‌ی خام + استخراج JSON با Gemini (لحظه‌ای)
    پایان روز → تجمیع رویدادها → محاسبه → اکسل → تلگرام + کانال بک‌آپ
```

**پشته:** Next.js 16 · TypeScript · grammY · Google Gemini · Drizzle ORM · Neon Postgres · ExcelJS

## 🚀 راه‌اندازی

### ۱) پیش‌نیازها را بگیرید (همه رایگان)

| مورد | از کجا |
|---|---|
| توکن بات تلگرام | [@BotFather](https://t.me/BotFather) → `/newbot` |
| کلید Gemini | [aistudio.google.com/apikey](https://aistudio.google.com/apikey) |
| دیتابیس Postgres | [neon.tech](https://neon.tech) (یا Vercel Postgres) |

> 💡 اگر در ایران هستید: ساختن کلید Gemini شاید یک‌بار به VPN نیاز داشته باشد، ولی
> در زمان اجرا چون درخواست‌ها از سرور Vercel (خارج ایران) زده می‌شوند مشکلی نیست.

### ۲) نصب و اجرای محلی (اختیاری)

```bash
npm install
cp .env.example .env.local   # مقادیر را پر کنید
npm run db:push              # ساخت جدول‌ها در دیتابیس
npm run dev
```

### ۳) استقرار روی Vercel

1. مخزن را روی Vercel وارد کنید (Import Project).
2. در **Settings → Environment Variables** این‌ها را اضافه کنید:
   `TELEGRAM_BOT_TOKEN` · `TELEGRAM_WEBHOOK_SECRET` · `GEMINI_API_KEY` ·
   `DATABASE_URL` · `PUBLIC_BASE_URL` · (اختیاری) `BACKUP_CHANNEL_ID` · `ALLOWED_CHAT_IDS`
3. Deploy کنید.
4. جدول‌ها را بسازید:
   ```bash
   npm run db:push
   ```
5. وبهوک تلگرام را ست کنید:
   ```bash
   npm run set-webhook
   ```

### ۴) استفاده

در تلگرام به بات پیام `/start` بدهید، سپس:

1. دکمه‌ی **▶️ شروع روز**
2. گزارش‌ها را بفرستید — مثلاً:
   - «علی ساعت ۸ اومد»
   - «اکیپ آرماتوربندی طبقه ۳ بودن، رضا و حسن»
   - «جوشکاری ستون‌های ضلع شمالی دوباره‌کاری شد چون تراز نبود»
   - (آخر روز) «علی ساعت ۸ رفت»
3. دکمه‌ی **⏹️ پایان روز** → فایل اکسل گزارش استاندارد ساخته می‌شود.
